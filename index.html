How to participate?

If you got a Bootstrap project hosted on GitHub,Bitbucket or Wordpress Plugins,you can add it to our feed following these steps:

  1. Fork the repository for the feed
  2. Add your project to feeds.json,alphabetize by project name
  3. Commit your changes and submit a pull request
All entries will be reviewed by a human being.
